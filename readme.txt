Add `[stream]` shortcode to WordPress for easy video sharing in your website

== Description ==

A really simple and very lightweight plugin to embed customized videos anywhere in your blog or website using wordpress shortcodes.
Stream Pandamolesto allows you to easily post [stream.pandamolesto.com](http://stream.pandamolesto.com) video to your site with [stream] shortcode.

###Plugin Usage###

Just copy your selected video id link in `[stream id="51112f94a" width="600" height="350" link="false"]` and set width and height of your player, if you want you can choose to change the link parameter to link="true" and below the player you can show the direct link of video in our website.

Simple example video: http://stream.pandamolesto.com/pandamolestocom/michelle-jenneke-si-riscalda-e-balla-agli-junior-world-championships-barcellona-2012-video_51112f94a.html

Select the univoque code after video_ (in this case 51112f94a) and copy between the stream id quotes.

You can also define video width and height with 'width' and 'height' and 'link' parameters. 


== Installation ==

1. Upload `pandamolesto-shortcode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


**See Also:** ["Installing Plugins" article on the WP Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)

###Plugin Activation###

Go to the admin area of your WordPress install and click on the "Plugins" menu. Click on "Activate" for the "Stream PandaMolesto Shortcode" plugin.
